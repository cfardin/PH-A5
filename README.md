- 1️⃣ What is the difference between var, let, and const?
- Ans : 
   - var :  
      - var is function scoped
      - can be re-declared and reassigned

   - let : 
      - let is blocked scoped 
      - can't be re-declared 
      - can reassigned like var 

   - const: 
      - blocked scoped like let
      - can not re-declared or reassigned
      - should be initialized when declared

- 3️⃣ What is the difference between map(), filter(), and forEach()?
-Ans : 
  -map() : for creating new array for updating or converting an array.
  -filter() : Creates a new array from an array with elements which match a condition
  -forEach() : Like a for loop .
